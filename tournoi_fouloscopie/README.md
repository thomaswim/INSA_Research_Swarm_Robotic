# Banc de Poisson

## Infos Arene et Robots

### Dimensions Arenes
3.2m x 2m

### Dimensions Robots
11cm de largeur et 10cm de long
Soit 110cm², approximons à un cercle de rayon de 11cm.
Distance minimale entre deux centres de robots est de 15 cm, soit 4 cm de débattement.
